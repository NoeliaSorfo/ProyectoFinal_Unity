El juego consiste en sortear una serie de obstáculos para llegar a la meta. Cada vez que seamos golpeados por un elemento danino o caígamos del mapa seremos devueltos al último punto de Respawn. Cuando lleguemos a la meta recibiremos el mensaje "You win".

Controles

Movimiento:
-Flechas

Saltar:
-Barra espaciadora

Correr:
-Shift izquierdo

Cambiar Cámaras:
-Enter
